Navigation Bar

If you want to make a simple navigation bar, you can use horizontal lists. Here is an example:

  • Home
  • Page 1
  • Page 2
  • Page 3
            <ul style="text-decoration:none">
            <li style="display:inline"><a href="#"">Home</a></li>
            <li style="display:inline"><a href="#"">Page 1</a></li>
            <li style="display:inline"><a href="#"">Page 2</a></li>
            <li style="display:inline"><a href="#"">Page 3</a></li>
            </ul>

Textarea

If you want to make a box that users can type in, this is the way to do it.

        <textarea>
        Premade text
        </textarea>

Code blocks

If you want to make a code block, there are two steps. First, put your html code you want to show in this converter. Then, put it into this code snippet:

        <pre>
        Put the converter results here
        </pre>
