Rainbow Text

With this, adding the rainbow-text class to a tag will make the text rainbow.


Rainbow

        <style>
            .rainbow-text {
                margin-top: 0px;
                margin-bottom: 10px;
                font-family: sans-serif;
                font-size: 6rem;
                background: linear-gradient(to right, #ef5350, #f48fb1, #7e57c2, #2196f3, #26c6da, #43a047, #eeff41, #f9a825, #ff5722);
                -webkit-background-clip: text;
                -webkit-text-fill-color: transparent;
            }
        </style>
        <p class="rainbow-text">Rainbow</p>
